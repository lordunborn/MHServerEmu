# Phantom Heroes — BiS Gear Data — Attribution

`PhantomBiSGear.json` contains best-in-slot gear loadouts used by the nemesis
loot system (rank-5 level-60 nemeses wear and drop their hero's BiS set).

These loadouts are **derived from community build guides** published at
**itembase.mhbugle.com** (the Marvel Heroes Omega "Item Base"), created by
**AlexBond**. Used with permission — AlexBond granted use of the Item Base build
data for this feature on the condition that the site is credited. Full credit for
the underlying builds also goes to the individual guide authors and the mhbugle
community. The data here is a transformed mapping of those public build
recommendations onto this server's item prototype refs.

This is a non-commercial fan project. Marvel Heroes and all item/hero names are
the intellectual property of Marvel / The Walt Disney Company (originally
Gazillion Entertainment). No game data files are redistributed here — item refs
resolve against the client data each user already owns.

Thank you to AlexBond and the Item Base community. If you're playing this fork,
consider visiting <https://itembase.mhbugle.com/> for the guides that made this
feature possible.
